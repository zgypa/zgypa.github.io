# Toni Magni landing page

## To build locally

    nix develop -c bundle exec jekyll serve

## Publishing changes

Merge changes into `main` and push. Hosting is done on Github.io on branch main.

## Updating CV

- CV is stored in `Toni_Magni_CV.pdf` and `Toni_Magni_CV.html`.
- Use zgypa/resume repo to update the CV and copy here.
- `cp ../resume/dist/resume-full.html ./Toni_Magni_CV.html`
- Update PDF version by opening the HTML with web browser, print to file and save as `./Toni_Magni_CV.pdf`
